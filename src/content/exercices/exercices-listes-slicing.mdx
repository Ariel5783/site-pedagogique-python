nombres = [15, 42, 8, 23, 4, 16, 9, 27, 50, 3]

premiers = nombres[:3]

# 2. Trois derniers éléments
derniers = nombres[-3:]
print("Trois derniers:", derniers)  # [27, 50, 3]

# 3. Nombres pairs
pairs = [x for x in nombres if x % 2 == 0]

indice = nombres.index(23)
print("Indice de 23:", indice)  # 3

nombres.sort()

print("Après ajout de 35:", nombres)

# 8. Insérer 7 au deuxième indice
nombres.insert(2, 7)
print("Après insertion de 7:", nombres)

- `append()` ajoute à la fin

# 1. Carrés de 0 à 9
carres = [x**2 for x in range(10)]
print("Carrés:", carres)  # [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

# 2. Pairs de 0 à 20
pairs = [x for x in range(21) if x % 2 == 0]

# 3. Mots longs
mots = ["python", "programmation", "objet", "données", "informatique"]
longs_mots = [mot for mot in mots if len(mot) > 6]
print("Mots longs:", longs_mots)
# ['programmation', 'informatique']

# 4. Nombre de mots par phrase
phrases = ["hello world", "python est puissant", "programmation orientée objet"]
nb_mots = [len(phrase.split()) for phrase in phrases]
print("Nombre de mots:", nb_mots)  # [2, 3, 3]
```

- Avec condition : `[expression for item in iterable if condition]`
- `range(10)` génère 0 à 9
- `len(mot)` pour la longueur
- `split()` sépare une chaîne en mots

texte = "anticonstitutionnellement"

# 1. [0,1,2,3,4,5]

# 2. 'anti'
print(texte[:4])

# 3. 'constitution'

# ou
print(chiffres[::-2][:5])
```

- `[start:stop]` : de start à stop (exclus)
- `[start:stop:step]` : avec un pas
- `[::-1]` : inverse complètement
- Index négatifs partent de la fin

```python
def chercher_mots(mots, lettre):
    """
    Retourne les mots contenant une lettre donnée
    
    Args:
        mots (list): Liste de mots
        lettre (str): Lettre à chercher
    
    Returns:
        list: Mots contenant la lettre
    """
    return [mot for mot in mots if lettre in mot]

def chercher_mots_pattern(mots, pattern):
    """
    Retourne les mots contenant un pattern
    
    Args:
        mots (list): Liste de mots
        pattern (str): Sous-chaîne à chercher
    
    Returns:
        list: Mots contenant le pattern
    """
    return [mot for mot in mots if pattern in mot]

# Tests
mots = ["chat", "chien", "oiseau", "éléphant"]

print(chercher_mots(mots, "e"))
# ['chien', 'oiseau', 'éléphant']

print(chercher_mots_pattern(mots, "oi"))
# ['oiseau']

print(chercher_mots_pattern(mots, "ch"))
# ['chat', 'chien']
```

**Barème (2 points)** :

- `chercher_mots_pattern()` : 1 point

**Points clés** :
- Opérateur `in` pour tester la présence
- Compréhension de liste pour filtrer
- Les deux fonctions sont très similaires

```python
import random
import string

def generer_mot_de_passe(longueur):
    """
    Génère un mot de passe aléatoire
    
    Args:
        longueur (int): Longueur du mot de passe
    
    Returns:
        str: Mot de passe généré
    """
    # Définir les caractères possibles
    caracteres = (
        string.ascii_lowercase +  # a-z
        string.ascii_uppercase +  # A-Z
        string.digits +           # 0-9
        "!@#$%^&*()"             # Spéciaux
    )
    
    # Générer le mot de passe
    mot_de_passe = ''.join(random.choice(caracteres) for _ in range(longueur))
    
    return mot_de_passe

# Tests
print(generer_mot_de_passe(12))
print(generer_mot_de_passe(8))
print(generer_mot_de_passe(20))

# Version alternative avec garantie d'au moins un de chaque type
def generer_mot_de_passe_secure(longueur):
    """Version avec garantie d'au moins un caractère de chaque type"""
    if longueur < 4:
        raise ValueError("La longueur doit être au moins 4")
    
    # Garantir au moins un de chaque
    mot_de_passe = [
        random.choice(string.ascii_lowercase),
        random.choice(string.ascii_uppercase),
        random.choice(string.digits),
        random.choice("!@#$%^&*()")
    ]
    
    # Compléter avec des caractères aléatoires
    caracteres = string.ascii_letters + string.digits + "!@#$%^&*()"
    mot_de_passe += [random.choice(caracteres) for _ in range(longueur - 4)]
    
    # Mélanger
    random.shuffle(mot_de_passe)
    
    return ''.join(mot_de_passe)
```

**Barème (2 points)** :
- Utilisation correcte de random : 0.5 point
- Inclusion de tous les types de caractères : 1 point
- Fonction opérationnelle : 0.5 point

**Points clés** :
- Module `string` pour les constantes
- `random.choice()` pour sélection aléatoire
- Compréhension de liste avec `join()`
- Version avancée garantit diversité
