## Introduction

Les variables sont des éléments fondamentaux de tout langage de programmation. En Python, une variable permet de stocker des informations en mémoire pour les réutiliser plus tard dans votre programme.

## Qu'est-ce qu'une variable ?

Une **variable** est un espace de stockage dans la mémoire de l'ordinateur, identifié par un nom, qui permet de conserver une valeur. On peut comparer une variable à une boîte étiquetée dans laquelle on range des informations.

### Déclaration d'une variable

En Python, déclarer une variable est très simple. Il suffit de choisir un nom et de lui assigner une valeur avec le signe égal `=` :

```python
age = 18
nom = "Alice"
taille = 1.75
est_etudiant = True
```

### Règles de nommage

Les noms de variables en Python doivent respecter certaines règles :

- Commencer par une lettre (a-z, A-Z) ou un underscore `_`
- Contenir uniquement des lettres, chiffres et underscores
- Ne pas être un mot-clé réservé de Python (`if`, `for`, `while`, etc.)
- Être sensibles à la casse (`age` et `Age` sont différents)

**Conventions de nommage (PEP 8) :**
- Utiliser des minuscules avec des underscores pour séparer les mots : `mon_age`, `nom_complet`
- Choisir des noms explicites et significatifs

## Les types de données de base

Python propose plusieurs types de données de base :

### 1. Les nombres entiers (int)

Les nombres entiers sont des nombres sans décimale :

```python
age = 25
annee = 2025
temperature = -5
```

### 2. Les nombres à virgule flottante (float)

Les nombres décimaux utilisent le point comme séparateur :

```python
taille = 1.75
prix = 19.99
temperature = -3.5
```

### 3. Les chaînes de caractères (str)

Les chaînes de caractères représentent du texte, encadré par des guillemets simples ou doubles :

```python
prenom = "Alice"
ville = 'Paris'
message = "Bonjour tout le monde !"
```

### 4. Les booléens (bool)

Les booléens ne peuvent prendre que deux valeurs : `True` (vrai) ou `False` (faux) :

```python
est_majeur = True
a_permis = False
```

## Les opérations sur les variables

### Opérations arithmétiques

Python supporte les opérations mathématiques classiques :

```python
a = 10
b = 3

addition = a + b        # 13
soustraction = a - b    # 7
multiplication = a * b  # 30
division = a / b        # 3.333...
division_entiere = a // b  # 3
modulo = a % b          # 1 (reste de la division)
puissance = a ** b      # 1000
```

### Opérations sur les chaînes

Les chaînes de caractères ont aussi leurs opérations :

```python
prenom = "Alice"
nom = "Dupont"

# Concaténation
nom_complet = prenom + " " + nom  # "Alice Dupont"

# Répétition
separateur = "-" * 20  # "--------------------"

# Longueur
longueur = len(prenom)  # 5
```

## La fonction `type()`

Pour connaître le type d'une variable, utilisez la fonction `type()` :

```python
x = 42
print(type(x))  # <class 'int'>

y = 3.14
print(type(y))  # <class 'float'>

z = "Python"
print(type(z))  # <class 'str'>
```

## Conversion de types

Il est possible de convertir une variable d'un type à un autre :

```python
# Convertir en entier
nombre = int("42")       # 42
arrondi = int(3.7)       # 3

# Convertir en float
decimal = float("3.14")  # 3.14
decimal2 = float(5)      # 5.0

# Convertir en chaîne
texte = str(42)          # "42"
texte2 = str(3.14)       # "3.14"
```

## Affichage avec `print()`

La fonction `print()` permet d'afficher des informations à l'écran :

```python
nom = "Alice"
age = 20

# Affichage simple
print(nom)

# Affichage multiple
print(nom, age)

# Formatage avec f-strings (Python 3.6+)
print(f"Je m'appelle {nom} et j'ai {age} ans.")

# Formatage avec .format()
print("Je m'appelle {} et j'ai {} ans.".format(nom, age))
```

## Résumé

- Les variables stockent des informations en mémoire
- Python propose 4 types de base : `int`, `float`, `str`, `bool`
- Les opérations dépendent du type de données
- La fonction `type()` identifie le type d'une variable
- La conversion de types se fait avec `int()`, `float()`, `str()`
- La fonction `print()` affiche des informations
