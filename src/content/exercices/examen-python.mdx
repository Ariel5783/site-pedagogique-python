### Question 3 : Factorielle (4 points)

**Solution 1 : Avec boucle**
```python
def fac(n):
    if n == 0:
        return 1
    resultat = 1
    for i in range(1, n + 1):
        resultat *= i
    return resultat
```

**Solution 2 : R√©cursive**

### Question 4 : PGCD (4 points)

```python

    while b != 0:
        a, b = b, a % b
    return a
```

**Bar√®me** :
- D√©finition correcte de la fonction : 0.5 point
- Utilisation de l'algorithme d'Euclide : 2.5 points
- Return du r√©sultat : 0.5 point
- Fonction test√©e et op√©rationnelle : 0.5 point

**Algorithme d'Euclide** :
1. Tant que b ‚â† 0
2. Remplacer (a, b) par (b, a mod b)
3. Retourner a

**Variante possible** :
