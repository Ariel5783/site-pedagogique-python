## Exercice 4 : Factorielle

```python
# Fichier : factorielle.py
def fac(n):
    if n == 0:  # Cas particulier
        return 1
    produit = 1
    for i in range(1, n + 1):  # Boucle pour multiplier les nombres
        produit *= i
    return produit

# Exemple d'utilisation
n = int(input("Entrez un nombre : "))
print(f"Factorielle de {n} = {fac(n)}")
```

**Explications** :
1. Une fonction `fac(n)` est définie
2. Cas particulier : 0! = 1
3. La boucle `for` multiplie chaque nombre jusqu'à `n`
4. Le résultat est retourné

**Barème (4 points)** :
- Définition de la fonction : 0.5 point
- Cas particulier n=0 : 1 point
- Boucle correcte : 1.5 point
- Return et tests : 1 point

## Exercice 5 : PGCD

```python
# Fichier : pgcd.py
def pgcd(a, b):
    while b != 0:  # Boucle jusqu'à ce que b devienne 0
        a, b = b, a % b  # Algorithme d'Euclide
    return a

# Exemple d'utilisation
a = int(input("Entrez a : "))
b = int(input("Entrez b : "))
print(f"PGCD({a}, {b}) = {pgcd(a, b)}")
```

**Explications** :
1. Utilise l'algorithme d'Euclide
2. On échange `a` et `b` jusqu'à ce que `b` soit 0
3. Le PGCD est alors la dernière valeur de `a`

**Barème (4 points)** :
- Définition de la fonction : 0.5 point
- Boucle while correcte : 1.5 point
- Algorithme d'Euclide (échange) : 1.5 point
- Return et tests : 0.5 point

## Exercice 6 : Pair ou Impair

## Exercice 7 : Maximum de deux nombres

```python
# Fichier : max_deux_nombres.py
a = int(input("Entrez le premier nombre : "))
b = int(input("Entrez le deuxième nombre : "))
maxi = a if a > b else b  # Opérateur ternaire

```

**Alternative avec if/else classique** :
```python
a = int(input("Entrez le premier nombre : "))
b = int(input("Entrez le deuxième nombre : "))
if a > b:
    maxi = a
else:
    maxi = b
print(f"Le maximum entre {a} et {b} est {maxi}.")
```

**Explications** :
1. Compare les deux nombres avec l'opérateur ternaire ou if/else
2. Affiche le plus grand nombre
