## Notes pédagogiques

### Difficultés courantes
- Confusion entre `=` (affectation) et `==` (égalité)
- Oubli des guillemets pour les chaînes
- Division entière vs division décimale
- Parenthèses dans les calculs de moyenne

### Points de vigilance
- Vérifier que les élèves testent leur code
- Insister sur la lisibilité (noms de variables, commentaires)
- Encourager l'utilisation des f-strings

### Prolongements
- Ajouter des exercices avec input() pour récupérer des valeurs
- Introduire les conditions (if/else)
